# Marketing AI Agents 🚀

Kuusi erikoistunutta Claude AI -agenttia markkinointiisi. Suomi/English.

## Agentit / Agents

| Agentti | Tehtävä |
|---------|---------|
| ✍️ Sisältökirjoittaja | Blogit, artikkelit, markkinointitekstit |
| 📱 Some-manageri | Instagram, Facebook, LinkedIn, TikTok |
| 🎯 Mainostekstit | Google Ads, Meta Ads, konvertoiva copy |
| 💎 Brändistrategisti | Identiteetti, ääni, arvolupaus |
| 🔍 SEO-analyytikko | Avainsanat, optimointi, sijoitukset |
| 📊 Analytiikka | Data, KPI:t, päätökset |

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
uvicorn app:app --reload
```

Open http://localhost:8000
